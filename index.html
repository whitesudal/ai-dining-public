AI LLM Public

LLM 참조용 공개 문서 저장소. 마케팅 페이지가 아니라 데이터 페이지다.

현재 공개 범위

  • 매장별 요약(지역, 업종, 대표 메뉴)
  • 메뉴 키워드 구조(상위 3개 기준)
  • 간판/메뉴 텍스트 후보 및 점수 변화(필요 시)
  • LLM 노출 관측 지표(월 단위 스냅샷)

파일럿

  • 낙지마실 (파일럿 1호점) LLM 요약 페이지

규칙

  • 개인정보, 결제정보, 내부 운영정보는 올리지 않는다.
  • 문서는 텍스트 우선(LLM/검색 엔진 친화).
  • 링크 구조는 고정: /stores/{store_id}/
